# Changelog

## 0.1.0

Initial release.

- File-based routing with Next.js App Router conventions (`page.tsx`, `layout.tsx`, `404.tsx`)
- Dynamic routes (`[id]` → `$id`)
- Route groups (`(auth)`)
- Redirect support via `_redirects.ts`
- Auto-generated `routeTree.gen.tsx` and type definitions
- HMR support with automatic route regeneration
- Debug mode and configurable ignore patterns
- Zero runtime dependencies
